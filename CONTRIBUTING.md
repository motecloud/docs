# Contributing

Thanks for helping improve MoteCloud documentation.

This repository accepts public documentation fixes, clarification requests, examples, and benchmark-methodology improvements. Product source code is developed outside this repository.

## Good Contributions

- Fix unclear wording, broken links, or outdated public references.
- Improve examples that use public APIs or public MCP workflows.
- Suggest missing concepts that help users understand MoteCloud.
- Clarify public benchmark methodology or result interpretation.

## Out Of Scope

Please do not submit tenant data, credentials, private repository links, internal runbooks, security evidence, private benchmark artifacts, or unpublished roadmap details.

## Review Checklist

Before opening a pull request, check that:

- Every GitHub link points to the `motecloud` organization.
- No private repository names or internal file paths are included.
- No secrets, tokens, tenant identifiers, or customer-specific data are present.
- Security material is limited to public disclosure process and safe best practices.
- Benchmark content is public methodology or published results only.
